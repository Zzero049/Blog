运用快排和找到最后一个峰值
```java
class Solution {
    public void nextPermutation(int[] nums) {
        int temp;
        int len = nums.length;
        int index=0;
        if(len>1){
            for(int i=1;i<len;i++){
                if(nums[i-1]<nums[i]){
                    index = i;
                }
            }
            if(index==len-1){
                temp = nums[index];
                nums[index] = nums[index-1];
                nums[index-1] = temp;
            }else if(index==0){
                quickSort(nums, 0, len-1);
            }else{
                int minCloseNum=nums[index];
                int minIndex=index;
                for(int i=index+1;i<len;i++){
                    if(nums[i]>nums[index-1] && minCloseNum>nums[i]){
                        minCloseNum = nums[i];
                        minIndex = i;
                    }
                }
                temp = nums[index-1];
                nums[index-1] = nums[minIndex];
                nums[minIndex] = temp;
                quickSort(nums, index, len-1);
            }
        }
    }

    public void quickSort(int[] nums, int start,int end){
        int i=start,j=end;
        int temp = nums[start];
        while(i<j){
            while(i<j&&nums[j]>temp) j--;
            if(i<j){
                nums[i] = nums[j];
                i++;
            }
            while(i<j&&nums[i]<=temp) i++;
            if(i<j){
                nums[j] = nums[i];
                j--;
            }
        }
        nums[i] = temp;
        if(start<i-1){
            quickSort(nums,start,i-1);
        }
        if(i+1<end){
            quickSort(nums,i+1,end);
        }
    }
}
```

官方解，异曲同工，但更精简，实际上并不需要排序，只需要倒置就可以了
```java
public class Solution {
    public void nextPermutation(int[] nums) {
        int i = nums.length - 2;
        while (i >= 0 && nums[i + 1] <= nums[i]) {
            i--;
        }
        if (i >= 0) {
            int j = nums.length - 1;
            while (j >= 0 && nums[j] <= nums[i]) {
                j--;
            }
            swap(nums, i, j);
        }
        reverse(nums, i + 1);
    }

    private void reverse(int[] nums, int start) {
        int i = start, j = nums.length - 1;
        while (i < j) {
            swap(nums, i, j);
            i++;
            j--;
        }
    }

    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```
